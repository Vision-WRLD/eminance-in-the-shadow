# Eminance in the Shadow — NOCTURNE

A dark-luxury fine-jewelry concept site. Multipage, vanilla HTML/CSS/JS, no build step. Blends the restraint of Tiffany & Co. with the editorial warmth of Van Cleef & Arpels, reimagined in obsidian and champagne gold.

## Pages
`index` · `collections` · `bespoke` · `craft` · `contact` · `checkout` · `privacy` · `terms` · `cookies`

## Features
- Looped video hero (local, watermark-free)
- Collections with live **sort** + **filter** drawer (price, type, material, size)
- Pandora-style product detail modal with **lens zoom**, metal swatches, per-piece story
- Persistent **favourites** + **bag** (localStorage) and a standalone **checkout** flow
- Full-screen **search**
- Even centered nav, custom motion, IntersectionObserver reveals
- Fully responsive (PC + mobile), reduced-motion aware
- Security-hardened: CSP, HSTS, COOP/CORP, Permissions-Policy, no cookies/trackers

## Run locally
```
python -m http.server 8678
```
Then open http://localhost:8678

## Stack
Vanilla HTML/CSS/JS. Fonts: Cormorant Garamond + Montserrat. Imagery: Unsplash. Video: royalty-free stock.

*Demonstration project. NOCTURNE is fictional; nothing is for sale.*
